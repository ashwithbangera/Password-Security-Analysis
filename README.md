# Task 4: Password Security & Authentication Analysis

## Objective
To understand how passwords are stored, attacked, and protected using secure authentication mechanisms.

## Tools Used
- John the Ripper
- Kali Linux

## Work Done
- Studied hashing and encryption concepts
- Generated MD5 password hash
- Cracked weak password using dictionary attack
- Analyzed password vulnerabilities
- Learned importance of Multi-Factor Authentication (MFA)

## Outcome
This task provided practical understanding of password security and common attack techniques.
